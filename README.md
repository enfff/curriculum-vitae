# Curriculum Vitæ

## Installazione

    python -m venv .venv    
    source .venv/bin/activate
    pip install rendercv[full]

## Uso

    source .venv/bin/activate

Modifica i file .yaml, e ricompila i file con

    rendercv render --watch Francesco_P._Carmone_CV.yaml

I file prodotti sono disponibili dentro `rendercv_output/`
